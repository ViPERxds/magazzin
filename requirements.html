Доплата при условии

Дополнительное вознаграждение, которое блогер получит при выполнении определенных условий. Например, если видео наберет определенное количество просмотров или лайков.

Выкуп по ключу

Блогер должен будет купить товар по промокоду или через специальную ссылку. Это подтвердит реальную покупку товара для обзора.

Возможность возврата

Если выбрано "Да", блогер сможет вернуть товар после съемки контента. При возврате сумма выкупа будет возмещена, а вознаграждение за контент сохранится.

Отзыв с фото

Блогер должен будет оставить отзыв о товаре на площадке с фотографиями. Это поможет повысить доверие к товару и увеличить продажи.

Как посчитать требования к блогеру

Укажите минимальные требования к статистике блогера для каждой соцсети. Это поможет найти подходящих блогеров для вашей рекламной кампании.

• Для Instagram важны показатели охвата Stories и просмотров Reels
• Для YouTube учитывайте просмотры на Shorts
• Для TikTok важен показатель среднего количества просмотров

Если какой-то показатель неважен, отметьте его галочкой "Неважно".
